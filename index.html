KUV плагины для Lampa

О проекте

Этот проект включает набор плагинов для Lampa, который позволяет подключать мультфильмы, студийные данные и стили, оптимизированные для различных платформ.

Плагин для мультфильмов

Плагин, который добавляет мультфильмы в ваш интерфейс, предоставляя информацию о каждом фильме.

https://example.com/plugin1

Плагин для студий

Плагин, который отображает информацию о студиях, использующих Lampa, и их фильмах/сериалах.

https://example.com/plugin2

Плагин для стилей

Этот плагин позволяет настраивать визуальные стили для Lampa, включая цветовую гамму, шрифты и иконки.

https://example.com/plugin3

Все плагины

Скачайте и подключите все плагины сразу, чтобы не подключать их по отдельности. Это удобный способ управлять всеми плагинами Lampa.

https://example.com/allplugins
